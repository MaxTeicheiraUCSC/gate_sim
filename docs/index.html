Data Processing

Simulation Pipeline

submit_jobs.sh └─ submit_gate.sbatch × N (parallel SLURM array) └─ czt_slit_simulation_cluster.py (each job) └─ output/job_XXXX/*.root merge_results.sbatch (depends on all sim jobs) ├─ merge_results.py → output/merged/ ├─ diagnostic_3d_plot.py → output/diagnostics/ ├─ analyze_phsp.py → output/analysis/ └─ Slack notification

Quick Commands
